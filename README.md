## 概况

支持接口自动化，web_ui自动化，app_ui自动化，未来将支持性能可靠性自动化，等其他功能

### 框架核心功能

1. 支持日志功能，日志能自动归档到执行目录
2. 支持报告功能，并支持钉钉等发送到通知群
3. 未来将具体支持各类配置，并提供示例
4. 未来将具体支持资源，并提供示例

### 接口自动化

1. 三层解耦，即数据层，逻辑层，业务层解耦
2. 支持函数式编程，代码简洁清晰，便于复制粘贴
3. 逻辑层的原理是，在api数据的基础上，增加入参解析填充功能，和响应信息提取的功能

### WEB_UI自动化

1. 支持执行时候自动截图，并归档到日志目录下
2. 支持PO模式，并提供示例

### APP_UI自动化

1. 支持PO模式，并提供示例

### 其他插件

1. 发送钉钉通知到通知群
