## 测试流程图

```mermaid
flowchart TD
    subgraph 创建流程
        C1[POST /system/dept] --> C2{deptName非空?}
        C2 -->|No| C3[返回错误: 必填项缺失]
        C2 -->|Yes| C4{deptName<=30字符?}
        C4 -->|No| C5[返回错误: 长度超限]
        C4 -->|Yes| C6{同父下名称唯一?}
        C6 -->|No| C7[返回错误: 名称已存在]
        C6 -->|Yes| C8{父部门状态正常?}
        C8 -->|No| C9[返回错误: 部门停用不允许新增]
        C8 -->|Yes| C10{phone<=11位?}
        C10 -->|No| C11[返回错误: 手机号长度超限]
        C10 -->|Yes| C12{email格式合法?}
        C12 -->|No| C13[返回错误: 邮箱格式错误]
        C12 -->|Yes| C14[创建成功<br/>设置ancestors=parent.ancestors+parentId]
    end

    subgraph 更新流程
        U1[PUT /system/dept] --> U2{parentId != deptId?}
        U2 -->|No| U3[返回错误: 上级部门不能是自己]
        U2 -->|Yes| U4{同父下名称唯一?}

        U4 -->|Yes| U6{status=1且有正常子部门?}

        U8 --> U12{status变为0?}

    end

    subgraph 删除流程
        D1[DELETE /system/dept/deptId] --> D2{存在子部门?}

        D2 -->|No| D4{部门下有用户?}
```
